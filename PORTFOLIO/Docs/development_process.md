# Development Process

This document explains the steps followed during the development of the portfolio website.

## 1. Planning & Wireframing
- Identified required sections: Hero, About, Skills, Projects, Contact.
- Planned layout structure on paper.
- Decided on a minimal and modern aesthetic.

## 2. Setting Up File Structure
Created the base folder and key files:
- 'index.html`
- `style.css'
- `assets/' for images and icons
- `docs/' for documentation

## 3. Writing the HTML Structure
- Built semantic HTML sections.
- Added a navigation bar with anchor links.
- Inserted placeholder project cards.

## 4. Designing with CSS
- Applied a color palette.
- Used Flexbox/Grid to make responsive layouts.
- Added spacing, shadows, and hover animations.
- Included media queries for mobile responsiveness.

## 5. Testing on Multiple Devices
- Previewed on desktop, tablet, and mobile.
- Adjusted typography for readability.
- Ensured all links and buttons function properly.

## 6. Final Polishing
- Compressed images for faster load times.
- Validated HTML/CSS using online validators.
- Cleaned code and added comments where necessary.
